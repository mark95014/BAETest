### How to run the tests
* **Visual Studio method**
  * Open Test->Test Explorer and run test or test case by right-clicking and choosing Run or Debug.
* **Command line method**
  * Open a Bash shell or DOS command prompt.
  * cd to .../BAETest/<project>
  * dotnet test

### Environment
* Set the test parameters in .runsettings
